Weather Dashboard Pro

A modern and responsive weather dashboard built using HTML, CSS, and JavaScript. The application provides real-time weather information, detailed weather statistics, and a clean user interface.

---

Features

Current Weather

- Search weather by city
- Current temperature
- Weather condition
- Weather icon
- Feels like temperature
- Humidity
- Wind speed
- Pressure
- Visibility
- UV Index
- Sunrise & Sunset
- Date & Time

User Experience

- Responsive design
- Modern glassmorphism UI
- Loading animation
- Error handling
- Keyboard support (Press Enter to Search)
- Dark Mode

Search Features

- Search any city
- Current location weather
- Recent search history
- Local Storage support

Utilities

- Copy weather report
- Download weather report
- Share weather report

---

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo API
- Open-Meteo Geocoding API
- Browser Geolocation API
- Local Storage API
